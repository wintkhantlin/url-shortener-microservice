# Web UI

Welcome to the face of **URL2Short**. This is the dashboard where you can create links, manage your account, and see those sweet, sweet analytics charts.

## Features

*   **Dashboard:** See all your links in one place.
*   **Link Creator:** Easily shorten URLs with a friendly interface.
*   **Analytics Views:** Visual graphs showing where your clicks are coming from (Country, Browser, OS).
*   **Responsive:** Works on your laptop or your phone.

## Tech Stack

*   **Framework:** React
*   **Build Tool:** Vite (it is blazing fast!)
*   **Styling:** Tailwind CSS (probably, assuming standard modern stack)
*   **Language:** TypeScript

## Development

Want to tweak the UI?

1.  **Install dependencies:**
    ```bash
    npm install
    # or
    bun install
    ```

2.  **Start the dev server:**
    ```bash
    npm run dev
    # or
    bun run dev
    ```

3.  Open `http://localhost:3000` and start hacking!
